# BovineLens: Cattle & Buffalo Breed Classification

BovineLens is a modern web application designed for the identification and classification of various cattle and buffalo breeds using AI.

## Features
- **Breed Identification**: Classify over 60 different breeds of cattle and buffalo.
- **Multilingual Support**: Available in multiple regional languages including Hindi, Tamil, Telugu, and more.
- **Structural Analysis**: Perform detailed structural measurements and evaluation.
- **History Tracking**: Keep track of previous analyses.

## Tech Stack
- **Frontend**: React, Vite, Vanilla CSS.
- **AI/ML**: YOLOv8 (weights exported to ONNX for web inference).
- **Backend Utilities**: Python scripts for data management and model training.

## Getting Started
1. Navigate to the `bovine-lens` directory.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
